## How to Import to Fastmail

### Option 1: Web Interface (Recommended)
1. Log into Fastmail ‚Üí Calendar
2. Click **Settings** (gear icon) ‚Üí **Import Calendar**
3. Select the file: `central_bank_meetings_2026.ics`
4. Choose destination calendar (or create new "Central Banks" calendar)
5. Click **Import**

### Option 2: Drag & Drop
1. Open Fastmail Calendar in browser
2. Drag the `.ics` file directly into the calendar view
3. Confirm import when prompted

### Option 3: Add to Existing Calendar
1. Fastmail will create a new calendar called "Central Bank Meetings" automatically
2. You can merge with existing calendar if preferred
