## 4. WACC Calculation

### Cost of Equity (CAPM)
- Risk‑free rate (10‑yr Treasury): **4.5%**
- Beta (levered, 5‑yr): **1.20**
- Market risk premium: **5.5%**
- **Re** = 4.5% + 1.20 × 5.5% = **11.1%**

### Cost of Debt
- Apple bond yield (10‑yr): **4.8%**
- After‑tax **Rd** = 4.8% × (1 – 0.15) = **4.08%**

### Capital Structure
- Market cap (E): **$2,980B**
- Net debt (D): **$40B**
- Enterprise value (V) = E + D = **$3,020B**
- Equity weight = 2,980 / 3,020 = **98.7%**
- Debt weight = 1.3%

### WACC
WACC = 98.7% × 11.1% + 1.3% × 4.08% ≈ **10.96% → 11.0%**

## 5. Terminal Value

### Method A – Perpetuity Growth (Gordon)
- FCFF_2030 = $140.1B
- Terminal growth (g) = **2.5%**
- TV = 140.1 × (1.025) / (0.11 – 0.025) = 143.6 / 0.085 = **$1,689B**

### Method B – Exit Multiple (EV/EBITDA)
- FY2030 EBITDA = Op. Income + D&A = 164.7 + 14.4 = **$179.1B**
- Peer‑group average EV/EBITDA = **18.0×** (conservative)
- TV = 179.1 × 18.0 = **$3,224B**

*Note: We use the Gordon method for base case to avoid double‑counting multiple expansion.*

## 10. Key Drivers & Risks

### Key Value Drivers
- iPhone install base replacement cycle
- Services revenue (App Store, iCloud, Apple TV+, Advertising) growth rate
- Gross margin trajectory (component costs, pricing power)
- Capital return program (buybacks, dividend)
- New product adoption (Vision Pro, AI‑enabled features)
- Regulatory outcomes (EU DMA, US antitrust)
- Foreign exchange rates

### Major Risks
1. **Smartphone saturation** – replacement cycles lengthening
2. **Services deceleration** – App Store growth slowing, privacy restrictions
3. **Margin compression** – component cost inflation, supply‑chain concentration
4. **Geopolitical** – China/Taiwan tensions affecting production and sales
5. **Regulatory** – fines, forced ecosystem changes, reduced commission rates
6. **Execution risk** – Vision Pro and AI features failing to gain adoption

## 11. Formulas Reference (for Excel rebuild)

```
NOPAT = Operating Income × (1 – Tax Rate)
FCFF = NOPAT + D&A – Capex – ΔNWC
ΔNWC = % of Revenue change × Revenue (minimal for Apple)

WACC = (E/V) × Re + (D/V) × Rd × (1 – Tc)
Re = Rf + β × MRP

PV_FCFF_t = FCFF_t / (1+WACC)^t
Terminal Value (Gordon) = FCFF_T × (1+g) / (WACC – g)
Terminal Value (Exit Multiple) = EBITDA_T × Multiple

Enterprise Value = Σ PV_FCFF + PV_Terminal
Equity Value = Enterprise Value – Net Debt
Value per Share = Equity Value / Shares Outstanding
```
